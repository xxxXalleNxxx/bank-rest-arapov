# Система управления банковскими картами

# Система аутентификации и авторизации

- Интеграция Spring Security с JWT-токенами через RBAC (ADMIN, USER)

- Настройка защищенных эндпоинтов в соответствии с требованиями безопасности

# Модуль управления картами

- Полный CRUD-функционал для банковских карт

- Реализация масок отображения номеров карт (формат: **** **** **** 1234)

- Управление статусами карт: Активна, Заблокирована, Истек срок

- Валидация данных и обработка ошибок

# Бизнес-логика для различных ролей

- Администратор: полный доступ к управлению картами и пользователями (CRUD)

- Пользователь: просмотр собственных карт, переводы между счетами, запросы на блокировку

# Инфраструктура проекта

- Настройка MySQL с использованием Liquibase миграций

- Контейнеризация через Docker Compose

- Реализация REST API с пагинацией и фильтрацией

- Интеграция Swagger/OpenAPI для документации API

- Мок тестирование ключевых элементов
